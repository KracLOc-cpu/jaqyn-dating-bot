# AGENTS.md — журнал для всех ИИ-агентов

> **Над проектом Jaqyn работает несколько ИИ-агентов (Codex, Claude и др.).**
> Этот файл — единая точка координации. Прочитай его в начале работы и
> **обязательно дополняй в конце своей сессии.**

## ⛏ Правила (читать перед работой)

1. **Любое изменение — записывай в раздел «Журнал изменений» ниже.** Одна строка =
   одно осмысленное изменение. Указывай: дату, кто ты (агент), что сделал, какие
   файлы затронул, статус (сделано / в работе).
2. **Новые записи — сверху** (свежее первым), под заголовком сессии с датой.
3. **Идеи на будущее** не реализуй сразу — добавляй в раздел «Идеи на будущее»
   со статусом `💡 предложено`. Реализовывать можно только после того, как
   **владелец проекта одобрит** (`✅ одобрено`).
4. Не дублируй: подробный технический дневник — в `PROGRESS.md`, план — в `PLAN.md`,
   деплой — в `DEPLOY.md` / `FREE_TEST_DEPLOY.md`. Здесь — **кто что менял** и **идеи**.
5. **Секреты и подключения:**
   - Карта сервисов (ссылки, дашборды, имена env) — `SERVICES.md` (коммитится).
   - Сами значения (пароли, токены, ключи) — `CREDENTIALS.local.md` (**только на ПК,
     в `.gitignore`, НЕ коммитить — репозиторий публичный!**).
   - Никогда не пиши секреты в журнал, код или любой коммитимый файл. Перед `git add`
     проверь `git status` — `CREDENTIALS.local.md` там быть не должно.

**Легенда статусов идей:** `💡 предложено` · `✅ одобрено` · `🚧 в работе` · `✔️ сделано` · `❌ отклонено`

## 🏗 Архитектура (v1 — решено 2026-06-18)

Принцип: **«почти $0 без хрупких костылей»**. Не трогаем работающую Python-логику и базу
перед запуском. Подробный расчёт и чек-лист запуска — в плане
`~/.claude/plans/recursive-floating-glacier.md`.

| Слой | v1 | Стоимость |
|------|----|-----------|
| Frontend | Cloudflare Pages (статичный Vite-билд) | $0 |
| Backend + бот | Render Starter, Python, always-on, бот через **webhook одним сервисом** | $7/мес |
| База | Supabase Postgres (Free → Pro когда данные жалко) | $0 → $25 |
| Фото | Cloudflare R2 (egress бесплатный) | $0 |
| Бэкапы | `pg_dump` по расписанию (GitHub Actions), gpg-шифрование | $0 |

**Cloudflare Workers — это v2-миграция, НЕ перед запуском** (потребовала бы переписать
FastAPI+aiogram+SQLAlchemy+boto3 на TS). Условия для v2: тесты на ленту/матчинг, реальная
нагрузка, причина сильнее «сэкономить $7».
